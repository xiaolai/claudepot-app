# Sample heartbeat email

---

```
Subject: Heartbeat — Dad's MacBook (2026-05-02)

Dad's MacBook is alive as of 2026-05-02 09:00.
```

---

The heartbeat is paired with the weekly report. If you stop
receiving heartbeats, something has gone wrong on the
dependent's machine — the agent stopped, the laptop is offline,
or the email pipeline broke.
