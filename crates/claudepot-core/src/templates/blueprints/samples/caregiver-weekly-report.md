# Sample of what your caregiver receives

This is what the email looks like. Plain text, one screen.

---

```
Subject: Weekly health report — Dad's MacBook (week ending 2026-05-02)

Weekly health report for Dad's MacBook
Period: 2026-04-25 to 2026-05-02

No specific concerns this week.

---
Disk: 64.1% used, 358.0 GB free (filling at ~1.2 GB/week)
Backups: Time Machine last completed 2026-05-02 06:00; iCloud in sync.
Updates: 2 macOS pending, 1 App Store pending.
Battery: 412 cycles; 87% of design capacity.
Login attempts: 14 total, 0 anomalous.
Crashes: 0.
```

---

What the email contains
- Disk space usage and trend
- Time Machine and iCloud status
- Pending macOS / App Store updates count
- Battery cycle count + capacity (laptops only)
- Login items / startup programs that appeared this week
- New apps installed this week
- Login attempt count + anomaly count
- Crash count + which apps crashed

What it never contains
- Files, file paths, file contents
- Browser history, bookmarks, downloads
- Messages, email contents, calendar
- Photos, location, application data
- The dependent's typed name (only the label "Dad's MacBook")
