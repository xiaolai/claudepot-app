# Install

> **Status: beta (`0.1.22`).** Daily-driven on macOS. Linux and
> Windows builds are green but less seasoned.
>
> **Pre-built installers are coming.** Until they ship, ClauDepot is
> source-build only. The build is fast (under two minutes on a
> modern laptop) and needs no system dependencies beyond the Rust
> and Node toolchains.

Source is at
[github.com/xiaolai/claudepot-app](https://github.com/xiaolai/claudepot-app).

## Prerequisites

- A recent **Rust toolchain** ([rustup.rs](https://rustup.rs)).
- **Node 20+** with **pnpm** ([pnpm.io](https://pnpm.io)).

That's it. No Xcode-only tooling on macOS, no MSVC on Windows beyond
what `rustup` already pulls in, no extra system libraries on Linux.

## Build

```bash
git clone https://github.com/xiaolai/claudepot-app
cd claudepot-app
pnpm install

# Command-line tool
cargo build -p claudepot-cli --release
# Binary lands at ./target/release/claudepot

# Desktop app
pnpm tauri build --no-bundle      # builds a binary; no installer
# OR
pnpm tauri dev                    # run the GUI in dev mode (hot reload)
```

`pnpm tauri build --no-bundle` skips the platform installer and
gives you the raw app bundle in
`src-tauri/target/release/bundle/`. On macOS that's a `.app`; on
Linux it's a binary; on Windows it's an `.exe`.

## First run

When you launch ClauDepot the first time, it will:

1. **Check for an existing Claude Code session.** If you've run
   `claude /login` on this machine, ClauDepot finds your current
   credentials and offers to register them as your first account.
2. **Open the Accounts tab.** This is your home base — add other
   accounts here, switch the CLI and Desktop slots independently.
3. **Set up the OS keychain entry.** ClauDepot uses a separate
   keychain item from Claude's own — see the [architecture
   note](https://github.com/xiaolai/claudepot-app/blob/main/.claude/rules/architecture.md)
   for why this split matters.

Your data lives at `~/.claudepot/`. Override with the
`CLAUDEPOT_DATA_DIR` environment variable if your home directory is
on a small SSD.
